## Quick Thesaurus
A small Windows-focused desktop thesaurus utility that provides fast synonym and antonym lookup using Merriam‑Webster's thesaurus pages.

## Features
- GUI built with DearPyGui
- Global hotkey (Ctrl+Alt+A) to show/hide the app
- Spell-check suggestions
- Caching
- Configurable window positioning

> Note: The project scrapes Merriam‑Webster public pages. Use responsibly and be aware that site layout changes may break the parser.

## Requirements
- Windows
- Python 3.8+ recommended
- See `requirements.txt` for Python package dependencies

## Quick start
From the project root run:

```powershell
python quickthesaurus.py
```

This will start the application. Press Ctrl+Alt+A to toggle the window visibilty.
